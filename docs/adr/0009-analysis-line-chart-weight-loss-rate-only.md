# 分析画面の折れ線グラフは WeightLossRate のみ表示する

PRD US#39 は「WeightLossRate と Crack タイミングを折れ線グラフで表示する」と記述しているが、折れ線グラフに表示する系列を WeightLossRate の1本に限定する。

WeightLossRate（単位: %、例: 14.5）と FirstCrack / SecondCrack（単位: 秒、例: 300）は次元が異なるため、Y 軸を共有すると双方のスケールが崩壊して実質読み取れないグラフになる。デュアル Y 軸は nivo で実現可能だが、自宅焙煎の「直感的な振り返り」という目的に対して読み取りコストが高い。Crack タイミングの推移は将来の拡張（別セクションや専用グラフ）として扱う。
